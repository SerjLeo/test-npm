## Installation

`npm install say-hello-bysl`

## Usage

Say "Hello" at any language you want! Just pass language abbreviation to function or get random "Hello"!

##### Available languages:
* en - _English_
* ru - _Русский_
* es - _Español_
* uk - _Український_
* gr - _Ελληνικά_
* it - _Italiano_
* cn - _中文_
* jp - _日本人_
* se - _Svenska_
* cz - _čeština_
* in - _हिंदी_
* fr - _Français_
* fi - _Suomalainen_
